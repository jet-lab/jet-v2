This crate documents the instructions used in the margin program of the jet-v2 repo. Handler functions are described for each instruction well as struct parameters (and their types and descriptions are listed) and any handler function parameters aside from parameters that exist in every instruction handler function. Parameters of events emitted for the purposes of data logging are also included.
