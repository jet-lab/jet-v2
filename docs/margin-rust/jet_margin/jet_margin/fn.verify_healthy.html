Verify that the account is healthy, by validating the collateralization ration is above the minimum.

verify_healthy.rs

This instruction does the following:

  1. Let account be the loaded margin account.

  2. Check if all positions for that margin account are healthy.

a. If there are unhealthy positions exist for this margin account, return False.

  1. Emit the VerifiedHealthy event for data logging (see table below).

  2. Return Ok(()).
