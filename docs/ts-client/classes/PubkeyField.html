fromArray

  • fromArray(values: any[]): undefined | LayoutObject

  • Inherited from BL.Layout.fromArray

    • Defined in node_modules/@solana/buffer-layout/lib/Layout.d.ts:134

    Create an object from layout properties and an array of values.

    NOTE This function returns undefined if invoked on a layout that does not return its value as an Object. Objects are returned for things that are a {@link Structure}, which includes {@link VariantLayout|variant layouts} if they are structures, and excludes {@link Union}s. If you want this feature for a union you must use {@link Union.getVariant|getVariant} to select the desired layout.

    Parameters

    • values: any[]

      an array of values that correspond to the default order for properties. As with {@link Layout#decode|decode} layout elements that have no property name are skipped when iterating over the array values. Only the top-level properties are assigned; arguments are not assigned to properties of contained layouts. Any unused values are ignored.

    Returns undefined | LayoutObject

    Function to create an Object into which decoded properties will be written.

    Used only for layouts that {@link Layout#decode|decode} to Object instances, which means:

    • {@link Structure}
    • {@link Union}
